# Vue Coding Style

> Extends `rules/common/coding-style.md` with vue guidance.

## Requirements

- Use Composition API, refs/reactive intentionally, computed for derived state, Pinia for app state, and typed props/emits.
- Keep modules cohesive and testable.
- Prefer explicit names over comments.
- Validate data entering from users, files, network, or environment.
