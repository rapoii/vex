# Web Coding Style

> Extends `rules/common/coding-style.md` with web guidance.

## Requirements

- Use semantic HTML, design tokens, responsive layouts, accessible focus states, and compositor-friendly motion.
- Keep modules cohesive and testable.
- Prefer explicit names over comments.
- Validate data entering from users, files, network, or environment.
